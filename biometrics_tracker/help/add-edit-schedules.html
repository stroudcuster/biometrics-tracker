Adding or Editing Schedules

After selecting the Add or Edit menu option, if you are tracking more than one person, you will be prompted to select the person whose schedules you want to work with. Following that, the GUI shown below will be displayed. If schedules were previously established for a person, they will be displayed in a list on the lower portion of the GUI. Double clicking on a list item will display that schedule's information in the prompts on the top portion of the GUI, where they can be modified. To the left of each list entry there is a checkbox labeled Del. Clicking a box will delete the associated schedule when the Save button is clicked.

To create a new schedule enter the Start Date and End Date for the schedule. For One-Time schedules, the Start Date is the only date on which an entry session will be triggered. Next, selection a Data Point Type for the entry session. In the interest of providing a manageable user interface, each entry session can accept a single entry for the specified Data Point Type. The Seq Nbr field is a display-only field. Each person's Schedules will be automatically assigned a series of sequence numbers so that each one can be uniquely identified in the database. The Note entry field allows you to associate a comment with the schedule. This note will be displayed on the entry GUI when it is triggered. Next, select the type of schedule (e.g. Hourly, Daily, etc.) you want to create by clicking one of the radio buttons displayed horizontally across the GUI. Clicking one of the radio buttons will display an entry GUI suited for the particular type of schedule you selected.

Enter the required information and click the Save button when you are finished.

To modify an existing schedule entry, double-click it's entry in the list at the bottom of the GUI. The Schedule Entry's properties will be displayed, including the GUI entry appropriate to the schedule's Frequency. Make the desired changes and click the Save button when you are finished.
